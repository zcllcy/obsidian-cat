Feed

PDFs, Markdown, text, links, or copied conversations. Wiki Cat queues each source, parses it, asks your model to curate it, then writes durable notes into your Obsidian wiki.

Flow

  1. Feed a source to the cat.
  2. Review queue status locally.
  3. Open the generated source note in Obsidian.
  4. Ask across the curated wiki.

Ask

Chat with the wiki, grounded in notes.


            

            

              
Retrieval

              
Hybrid lexical search.

              
Questions are matched against Obsidian Markdown using English terms, CJK n-grams, title weighting, path weighting, and a small bilingual scientific synonym map. The answer cites the retrieved notes.
